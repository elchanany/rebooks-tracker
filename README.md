# מערכת מעקב ספרים (Rebooks Tracker)

ברוך הבא למערכת הענן שלך למעקב אחר ספרים ב"סיפור חוזר"!
המערכת הזו תרוץ 24/7 באופן חינמי לגמרי על השרתים של גיטהאב.

## מה הלאה? מדריך הפעלה קצרצר:

### 1. פתיחת הבוט בטלגרם
1. פתח את אפליקציית טלגרם בטלפון.
2. חפש את **@BotFather**.
3. שלח לו את ההודעה: `/newbot`
4. תן לבוט שלך שם (למשל `RebooksTrackerBot`).
5. תן לבוט שם משתמש חוקי (חייב להסתיים במילה bot, למשל `MyRebooks_bot`).
6. ה-BotFather ישלח לך בחזרה הודעה ארוכה עם **Token** סודי (נראה בערך ככה: `123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11`). תשמור אותו!

### 2. קבלת ה-Chat ID שלך
1. חפש בטלגרם את הבוט: **@userinfobot**.
2. שלח לו הודעה `/start`.
3. הוא יחזיר לך את ה-**Id** שלך (מספר ארוך כמו `123456789`). תשמור גם אותו!

### 3. העלאת הקבצים ל-GitHub
1. פתח חשבון ב-[GitHub](https://github.com/).
2. לחץ על הוספת פרויקט חדש (New Repository). קרא לו `rebooks-tracker` וסמן אותו כ-**Public** (ציבורי).
3. פתח את התיקייה במחשב שלך (`C:\Users\elchanan yehuda\Documents\FindBook`).
4. בממשק של גיטהאב, לחץ על "uploading an existing file" (להעלות קבצים קיימים).
5. גרור את כל הקבצים שבתיקייה לתוך החלון של גיטהאב ולחץ על Commit Changes.

### 4. הגדרת ההתראות בגיטהאב
כדי שהרובוט יוכל לשלוח לך הודעה לטלגרם, צריך להכניס את הסודות:
1. בתוך הפרויקט שלך בגיטהאב, היכנס ללשונית **Settings** (הגדרות).
2. בתפריט השמאלי רד למטה ולחץ על **Secrets and variables** -> ואז **Actions**.
3. לחץ על הכפתור הירוק **New repository secret**.
4. בשם (Name) כתוב בדיוק: `TELEGRAM_BOT_TOKEN` ובתמליל (Secret) הדבק את הטוקן שקיבלת מה-BotFather. לחץ על Add secret.
5. עשה זאת שוב: לחץ על **New repository secret**. בשם כתוב: `TELEGRAM_CHAT_ID` ובתמליל הדבק את מספר ה-ID שלך מה-userinfobot. לחץ על Add secret.

### 5. הפעלת ה-Dashboard (דף האינטרנט שלך)
1. שוב תחת ה-**Settings** של הפרויקט, רד למטה בתפריט השמאלי ל-**Pages**.
2. תחת "Build and deployment", ב-Source בחר ב-**Deploy from a branch**.
3. ב-Branch בחר ב-`main` (או `master`) ולחץ Save.
4. תוך דקה-שתיים, יופיע לך קישור לדף האינטרנט המעוצב שלך למעלה באותו עמוד!

**זהו! עכשיו הרובוט ירוץ כל 10 דקות ויעדכן את הדף שלך, וברגע שספר חזר למלאי תקבל מיד הודעה לטלגרם עם קליק אחד לקנייה.**
